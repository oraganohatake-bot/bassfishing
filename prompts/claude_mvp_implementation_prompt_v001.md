# Claude MVP Implementation Prompt v001

## プロジェクト概要

2Dリアル系バス釣りシミュレーターを開発する。

見た目はSFC時代風のシンプルな2Dドットゲーム。

しかし内部シミュレーションはリアル寄りで、

- 地形
- 水深
- ストラクチャ
- 魚AI

によって釣果が決まる。

最終目標は大規模シミュレーターだが、
今回はMVP（動作確認版）を実装する。

---

# MVP目標

プレイヤーが

```text
歩く
↓
釣り場へ行く
↓
キャストする
↓
魚を釣る
```

まで完成させる。
