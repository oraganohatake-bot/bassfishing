# Lake World Design v001

## コンセプト

本作の湖はランダム生成されるマップではない。

湖そのものは永続的に存在する固定ワールドであり、
プレイヤーは何年も同じ湖へ通いながら知識を蓄積していく。

魚の居場所は変化するが、
地形そのものは変化しない。

本作の基本思想は、

「地形は記憶できる。魚は毎日変わる。」

である。

# 設計方針

プレイヤーに数値を大量表示しない。

裏側でシミュレーションを行い、

プレイヤーは景色や経験から湖を理解していく。

本作は魚を探すゲームではなく、

「湖を攻略するゲーム」

である。
