# Progression Design v001

## コンセプト

本作は単なる釣果ゲームではない。

プレイヤー自身が成長し、

- 湖を読む
- 地形を覚える
- 魚の行動を理解する

ことで攻略していく。

RPGのレベルアップではなく、

プレイヤー知識の成長を重視する。

---

# 設計方針

本作の成長は数値レベルではない。

プレイヤー自身が経験を積み、

湖を理解することが最大の成長となる。

最終目標は、

「この湖のことなら誰よりも知っている」

という感覚をプレイヤーに与えることである。
