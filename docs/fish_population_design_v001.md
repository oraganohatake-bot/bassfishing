# Fish Population Design v001

## コンセプト

本作の魚は無限スポーンしない。

湖には実際に魚が生息しており、
時間の経過とともに成長し、
産卵し、
老衰し、
世代交代を繰り返す。

ただしゲームとしての快適性を維持するため、
全個体をシミュレーションするのではなく、
サイズによって管理方法を変更する。

本作の目的は、

「魚を生成すること」ではなく、

「湖という生態系を維持すること」

である。

# 設計方針

魚は無限に湧かない。

しかし絶滅もしない。

湖は常に生態ピラミッドを維持しながら、
世代交代を繰り返す。

プレイヤーは数字を管理するのではなく、

「この湖には本当に魚が生きている」

と感じられることを目標とする。
