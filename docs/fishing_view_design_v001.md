# Fishing View Design v001

## コンセプト

釣りビューは魚探画面ではない。

プレイヤーは実際に岸辺へ立ち、
湖を見ながら釣りをしている感覚を体験する。

見える情報は限定し、
見えない情報を推測することがゲーム性となる。

本作のテーマは、

「魚を見るゲームではなく、湖を読むゲーム」

である。

---

# 設計方針

プレイヤーに与える情報は最小限。

内部シミュレーションは複雑でも、
画面はシンプルに保つ。

見た目はSFC。

中身は本格バスフィッシングシミュレーター。

それが本作の方向性である。
