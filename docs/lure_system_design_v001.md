# Lure System Design v001

## コンセプト

本作のルアーは単なる装備品ではない。

同じルアーでも、

- 投げる場所
- 通すレンジ
- 巻く速度
- ロッド操作

によって結果が変化する。

プレイヤーはルアーを選ぶだけでなく、

どう操作するかを考える必要がある。

# 設計方針

プレイヤーはルアーを選ぶだけでは勝てない。

どこへ投げるか

どのレンジを通すか

どのように操作するか

が釣果を決定する。

本作は装備ゲームではなく、

「操作で魚を騙すゲーム」

である。
