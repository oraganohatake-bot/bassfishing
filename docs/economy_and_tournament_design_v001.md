# Economy And Tournament Design v001

## コンセプト

魚を釣ることで報酬を獲得し、

- 新しいルアー
- 新しいタックル
- ボート
- 魚探
- 大会参加

へ繋げていく。

プレイヤーは釣果によって活動範囲を広げ、
より大きな魚と大会へ挑戦していく。

# 設計方針

魚を釣ることが目的ではない。

魚を釣ることで活動範囲を広げ、

知識を蓄積し、

より大きな舞台へ挑戦していく。

プレイヤーは最終的に

「湖を知り尽くしたプロアングラー」

を目指す。
