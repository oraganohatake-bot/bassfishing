# Mobility Design v001

## コンセプト

移動手段は最小限。

プレイヤーは自らの足で湖を探索する。

新しい移動手段は、
より多くのポイントへアクセスするためのものとする。

# 設計方針

本作の移動手段は
快適化ではなく、

「新しい釣り場へのアクセス手段」

として設計する。

移動手段を手に入れるたびに、
新しい景色と新しい魚が待っている状態を目指す。
