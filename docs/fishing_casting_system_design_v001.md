# Fishing Casting System Design v001

## 目的

バス釣りにおける「正確なキャスト」の重要性をゲームの中心要素にする。

プレイヤーは単に遠くへ飛ばすのではなく、狙ったスポットへ正確にルアーを送り込む必要がある。

------------------------------------------------------------------------

## 基本操作

### キャスト

-   左クリック押下：キャスト開始（振りかぶり）
-   左クリック長押し：キャストパワー蓄積
-   左クリック離す：リリース

------------------------------------------------------------------------

## キャストゲージ

ゲージは時間経過で上昇する。

``` text
0% → 25% → 50% → 75% → 100%
```

理想リリースゾーンを設定する。

``` text
------ PERFECT ------
```

### PERFECT

-   飛距離：最大
-   精度：最大
-   着弾誤差：±0〜0.5セル

### GOOD

-   飛距離：良好
-   精度：良好
-   着弾誤差：±1〜2セル

### EARLY

-   飛距離不足
-   ショートキャスト

### LATE

-   オーバーキャスト
-   精度低下
-   着弾誤差増加

------------------------------------------------------------------------

## アキュラシー判定

キャスト結果は以下で決定する。

``` text
タイミング精度
×
ロッド性能
×
プレイヤースキル
```

------------------------------------------------------------------------

## スポット補正

大型魚は複数要素が重なるエリアに付きやすい。

例：

``` text
沈み木
+
ブレイク
+
ウィード
```

PERFECTキャスト成功時：

``` text
大型魚反応率 +50%
```

ミスキャスト時：

``` text
小型魚中心
```

------------------------------------------------------------------------

## 将来拡張

### ロッド性能

-   Accuracy
-   Power

高級ロッドほどPERFECTゾーンが広くなる。

### プレイヤースキル

``` text
casting_skill
```

成長に応じてPERFECT判定幅を拡大。

------------------------------------------------------------------------

## 設計思想

地形を読む ↓ 正確に投げる ↓ ルアーを選ぶ ↓ アクションで食わせる

本作においてキャストは単なる入力ではなく、釣果を左右する重要な技術要素とする。
