# Day Cycle Design v001

## コンセプト

本作において時間帯は単なる見た目の変化ではない。

同じポイントでも、

- 朝
- 昼
- 夕方

で魚の居場所が変化する。

プレイヤーは季節だけでなく、

「今が何時なのか」

を考えながら行動する必要がある。

# 設計方針

魚は常に同じ場所にいない。

同じポイントでも、

朝と昼と夕方では
全く違う釣りになる。

プレイヤーは湖だけでなく、

時間も攻略する必要がある。
