# Rod Control And Reeling Input Design v001

## 目的

釣り操作をシンプルに保ちながら、実際の釣りに近い操作感を実現する。

本作では、操作役割を明確に分ける。

```text
左手：ロッド操作
右手：リール操作
```

プレイヤーは複雑なボタン操作を覚えるのではなく、

- ロッドを動かす
- リールを巻く
- テンションを感じる

ことで釣りを行う。

---

## 基本方針

ロッド操作とリーリング操作を分離する。

```text
十字キー / 方向キー：ロッド
マウス左クリック：リール
SPACE：フッキング補助
```

将来的にはフッキングもロッド操作へ統合可能。

---

# ロッド操作

## 入力

```text
↑：ロッドを下げる
↓：ロッドを立てる
←：ロッドを左へ倒す
→：ロッドを右へ倒す
```

上下軸は反転操作とする。

---

## ↑入力

ロッドを下げる。

効果：

```text
テンションを抜く
ルアーを沈める
フォールさせる
ファイト時に魚を走らせる
```

---

## ↓入力

ロッドを立てる。

効果：

```text
しゃくる
テンションを掛ける
ルアーを跳ね上げる
フッキング
魚を浮かせる
```

---

## ← / → 入力

ロッドを左右へ倒す。

効果：

```text
ライン角度を変える
リトリーブコースを微調整する
魚の向きを変える
障害物から魚を離す
```

---

# ロッド移動速度

ロッドは入力方向へ約1秒以内で移動完了する。

推奨：

```text
0.8〜1.0秒で最大位置到達
```

リアルすぎる遅さではなく、ゲームとして反応しやすい速度にする。

---

## 入力強度

短押し：

```text
小さなトゥイッチ
```

中押し：

```text
ジャーク
```

長押し：

```text
リフト
テンション維持
```

---

# ボタンを離した時の挙動

方向キーを離すと、ロッドは自動的に中立位置へ戻る。

```text
入力
↓
ロッドが動く
↓
ボタンを離す
↓
中立位置へ戻る
```

ロッドを固定位置へ置く操作ではなく、
ロッドを動かして戻す操作として扱う。

---

## 中立位置

通常状態のロッド角度。

```text
軽く斜め上に構えた状態
```

例：

```text
     ／
    ／
   ／
```

---

# 動的ロッド制御

内部的には入力値と描画値を分ける。

```python
rod_input_x
rod_input_y

rod_visual_x
rod_visual_y
```

入力中：

```python
rod_input_y = 1.0
```

離した時：

```python
rod_input_y = 0.0
```

描画側：

```python
rod_visual_y += (rod_input_y - rod_visual_y) * return_speed
```

これにより、押した時は素早く動き、離すと自然に戻る。

# ロッドアクション判定

ロッドアクションは「位置」ではなく「移動速度」で判定する。

重要なのは、ロッドがどこにあるかではなく、
どれだけ素早く動いたかである。

---

## 小トゥイッチ

短く入力してすぐ離す。

```text
↓
離す
```

効果：

```text
小さくルアーが跳ねる
```

---

## ジャーク

強めに素早く入力。

```text
↓↓
離す
```

効果：

```text
大きくルアーが飛ぶ
リアクションバイトを誘発
```

---

## リフト

長く入力を維持。

```text
↓↓↓↓
```

効果：

```text
ルアーを持ち上げる
ジグ・ワーム向き
```

---

## フォール

ロッドを下げる、またはテンションを抜く。

```text
↑
```

効果：

```text
ルアーを沈める
自然落下
食わせの間を作る
```

---

## ライン角度調整

左右入力。

```text
←
→
```

効果：

```text
ルアーの通過コースを微調整
魚とのファイト方向を調整
```

---

# リール操作

## 基本操作

マウス左クリックでリールを操作する。

---

## 長押し

通常リーリング。

```text
左クリック長押し
```

効果：

```text
ルアーを一定速度で回収
泳層を安定させる
```

---

## ワンタップ

チョイ巻き。

```text
左クリック短押し
```

効果：

```text
少しだけラインを回収
ワームやジグの微調整に有効
```

---

## 連打

早巻き。

```text
クリック連打
```

効果：

```text
高速回収
リアクション狙い
ルアー回収
```

---

# ロッドとリールの役割分担

## リール

```text
ルアーを進める
ラインを回収する
泳層を維持する
```

## ロッド

```text
ルアーに変化を与える
テンションを管理する
魚を掛ける
魚を制御する
```

---

# ファイト時の操作

## 魚が走る

ロッドを魚の方向へ倒す。

```text
← / →
```

ライン角度を合わせてテンションを安定させる。

---

## 魚が潜る

```text
↓
```

ロッドを立てて浮かせる。

---

## ラインブレイク危険

```text
↑
```

ロッドを下げてテンションを逃がす。

# 設計思想

プレイヤーは「アクション名」を選ぶのではない。

ロッドを動かし、
リールを巻き、
ラインテンションを感じながら釣る。

本作の釣り操作は、

```text
左手でロッド
右手でリール
```

という実釣感を重視する。

ボタンを離すとロッドが中立へ戻ることで、
トゥイッチ、ジャーク、リフト、フォールが自然に発生する。

最終的には、少ない操作で深い釣り表現を実現する。
