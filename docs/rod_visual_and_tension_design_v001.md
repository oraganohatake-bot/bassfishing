# Rod Visual And Tension Design v001

## 目的

ロッドを単なる静止画像ではなく、ゲーム内の情報表示装置として扱う。

プレイヤーはロッドのしなりを見ることで、

-   テンション
-   魚の強さ
-   ラインブレイク危険度

を直感的に把握できる。

------------------------------------------------------------------------

## 基本方針

ロッドは固定スプライトではなく、動的描画で表現する。

Pygame上で複数の制御点を持つ折れ線として描画する。

``` text
Grip
 ↓
Butt
 ↓
Belly
 ↓
Tip
```

推奨:

-   5〜7制御点
-   折れ線描画
-   ピクセルアート風表現

------------------------------------------------------------------------

## 通常状態

ほぼ直線。

``` text
────────╮
         │
```

------------------------------------------------------------------------

## キャスト

### 振りかぶり

後方へしなる。

``` text
╭──────
│
```

### リリース直前

最大負荷。

``` text
╭────
│
╰─
```

### リリース後

前方へ戻る。

``` text
──────╮
      │
```

------------------------------------------------------------------------

## リトリーブ

軽いテンション。

``` text
─────╮
```

魚がいない状態でも微小なしなりを持つ。

------------------------------------------------------------------------

## バイト

魚がルアーに触れた瞬間。

瞬間的にティップが入る。

``` text
────╮
    ╰─
```

------------------------------------------------------------------------

## フッキング成功

強い入力。

``` text
───╮
   ╰──
```

プレイヤーへ即座に視覚フィードバックを与える。

## テンションシステム

内部値

``` python
line_tension
```

範囲

``` python
0.0 ～ 1.0
```

ロッド描画へ反映。

``` python
bend_amount =
    line_tension
    * rod_flex
```

## 設計思想

ロッドはUIの一部である。

数字ではなく、

「しなり」を見るだけで状況が分かることを目指す。

プレイヤーはロッドの曲がり方から魚の存在や強さを感じ取る。

最終的には、画面を見ただけで

「これは大物だ」

と分かる状態を目標とする。
