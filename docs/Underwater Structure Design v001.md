# Underwater Structure Design v001

## コンセプト

魚はランダム配置しない。

魚は環境条件によって集まる。

本作では

「変化の重なり」

が魚の付き場を決定する。

---

# スポット評価システム

水中マップの各セルは
Fish Holding Score を持つ。

---

# 基本ルール

変化なし

```text
フラット

魚は通過するだけ
```

---

変化1つ

```text
小型中心
```

```text
岩だけ

ウィードだけ
```

---

```text
中型が付きやすい
```

```text

沈み木 + シェード

```

# 設計方針

本作では

「どこに投げても同じ」

を作らない。

大型魚は必ず

複数の条件が重なった場所に存在する。

プレイヤーは景色と経験から
その場所を発見していく。
