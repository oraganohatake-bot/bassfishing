# Fish AI Design v001

## コンセプト

本作のバスは決められた巡回ルートを持たない。

環境変化に反応しながら行動する。

プレイヤーは魚そのものではなく、

- 水温
- 天候
- ベイト
- 地形
- 時間帯

を読み、

魚がどこにいるかを推測する。

---

# AI設計思想

バスに

```text
春モード
夏モード
秋モード
冬モード
```

は存在しない。

代わりに、

```text
快適
餌
安全
```

を評価して行動する。

その結果として、

春はシャロー

夏はディープ

秋はベイト追尾

冬はチャネル

という行動が自然発生する。

# 設計方針

魚はスポーン地点に固定しない。

環境に反応して移動する。

プレイヤーが感じるべきものは、

「魚を見つけた」

ではなく、

「魚がそこにいる理由が分かった」

である。
